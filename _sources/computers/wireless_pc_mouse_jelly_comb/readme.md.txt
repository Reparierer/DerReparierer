# Wireless PC mouse Jelly Comb button repair

![](figures/overview.png)

Manufacturer: `Jelly Comb`    
Type: Mouse `KS15BS-3` from set `WGJP-015BS`

## Description of failure
The left mouse button only responds sometimes. 
The other buttons (right and middle mouse buttons) continue to function as usual.

## Failure investigation
The mouse is easy to take apart. 
The button can be located and replaced directly. 
The replacement part can be found by searching online.
`Kailh Mute Button 6mm x 6mmm x 7.3mm`
