# Hand blender

![](figures/overview.png)

Manufacturer: `ESGE`    
Type: `Zauberstab M150`    

## Description of failure
The blender no longer turns properly.

## Failure investigation
The mixer is opened. 
The damage is immediately visible: 
the rubber coupling is broken and has spread throughout the device. 
It needs to be replaced.
Furthermore, the rubber cone also looks damaged and should be replaced.

Manufacturer material numbers:   
 * 420041 rubber coupling
 * 720020 washer
 * 783001 rubber cone
 * 420021 drive shaft
